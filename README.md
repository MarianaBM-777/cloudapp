# ☁️ CloudApp - Aplicación Web Interactiva
# Objetivo. 
Desarrollar una aplicación web interactiva que permita practicar:
- Estructuración de interfaces web con HTML5
- Diseño visual moderno con CSS
- Manipulación del DOM con JavaScript
- Formularios dinámicos e interactivos
- Navegación entre secciones
- Validación de respuestas en tiempo real
